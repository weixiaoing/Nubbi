# 首页最近编辑列表

## 功能目的

首页的“最近编辑”模块用于横向展示当前用户最近修改过的笔记，帮助用户快速回到近期工作内容。

## 用户可见行为

- 最近编辑卡片横向排列，卡片宽度固定，点击卡片进入对应笔记详情页。
- 当内容超出可视区域时，左右两侧显示渐变和翻页按钮。
- 横向翻页按完整卡片槽位移动，避免右侧出现半张文档卡片。
- 右侧渐变会从最后一张完整可见卡片之后开始，保证卡片本体先完整显示，右侧剩余区域再承接透明渐变效果。

## 关键技术设计

- `RecentNoteList` 通过 DOM 测量卡片宽度、列表 gap、容器宽度和列表左侧偏移，计算当前视口能完整容纳的卡片数量。
- 最大滚动偏移量按 `最大起始卡片索引 * 单张卡片步长` 计算，而不是直接使用 `scrollWidth - wrapperWidth`。
- 右侧渐变宽度根据完整卡片占用后的剩余宽度动态计算，并设置最小宽度，保证渐变区域稳定。
- 左右翻页按钮使用相同的卡片步长移动，偏移量始终夹在 `0` 到 `maxOffset` 之间。

## 边界和注意事项

- 如果最近编辑笔记数量不足以横向溢出，则不显示右侧翻页渐变。
- 如果视口宽度变化，组件会重新计算完整卡片数量、最大滚动偏移量和右侧渐变宽度。
- 卡片宽度和列表 gap 来自实际 DOM 样式，后续调整卡片样式时不需要同步修改滚动常量。
