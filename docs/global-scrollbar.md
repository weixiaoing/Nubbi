# 全局滚动条样式

## 功能目的

统一应用内滚动条的视觉效果，让主内容区、侧边栏和弹窗滚动区域保持更轻、更接近原生应用的表现。

## 用户可见行为

- 滚动条默认显示为细窄的浅灰色圆角条。
- hover 或拖动滚动条时，thumb 会轻微加深。
- 横向滚动条和纵向滚动条使用同一套尺寸、圆角和颜色。
- 不再显示传统滚动条按钮，视觉上更接近现代桌面应用。

## 关键技术设计

- 样式集中在 `client/src/index.css` 的 Tailwind base 层。
- 使用 CSS 变量定义滚动条尺寸、轨道颜色和不同交互状态下的 thumb 颜色。
- WebKit 浏览器通过 `::-webkit-scrollbar` 系列伪元素控制外观。
- Firefox 通过 `scrollbar-width` 和 `scrollbar-color` 获得接近效果。

## 注意事项

- 选择器排除了 `.scrollbar-none`，因此局部需要隐藏滚动条的区域仍可继续使用现有工具类。
- 滚动条可交互区域保留 12px，实际可见 thumb 通过 3px 透明边框收窄到约 6px，兼顾可点击性和轻量视觉。
